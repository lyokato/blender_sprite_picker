# Changelog

All notable changes to Sprite Sheet Material Picker are documented in this file.

## [1.0.0] - 2026-06-05

Initial release.

### Added

- Blender material add-on for selecting cells from a sprite sheet image.
- Material property integration through `material.sprite_sheet_settings.sprite_index`.
- Sprite sheet image selection and cell size controls.
- Shader node setup for sprite sheet UV mapping.
- Thumbnail-based cell picker with paged previews.
- Constant-interpolation keyframe insertion for `sprite_index`.
- Background add-on tests for Blender 4.5 and 5.1.
- UI smoke test for the Material Properties panel.
